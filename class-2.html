Class and id seletors in CSS

When assigning a class to your HTML element (<element class="class">), you can later reference it in your CSS by simply using .class{style:what-i-want}. You can also use an id in your html element (<element id="id">), and reference it in your CSS by using #id{style:what-i-want}.
